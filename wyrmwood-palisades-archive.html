``` Here's a breakdown of every new component and the design thinking behind each: **Archive Banner** — A flex row with the page title on the left and a three-cell stat block on the right. The title uses `UnifrakturMaguntia` in `--teal-lt` at a size between the site title and a section header, so it reads as a page heading without competing with the main `

`. The stat block uses the same double inset-border treatment as the featured panels, with `Cinzel Decorative` numerals in `--red-pale` for immediate scannability. **Filter Bar** — Sits flush above the archive list with `border-bottom: none` so it reads as a unified block. Each tab carries the category's FA icon in `--amber` alongside the Cinzel label, and a small count badge in a bordered pill. The active tab gets `border-bottom: 2px solid var(--teal-lt)` — the only place in the design system where a bottom accent line is used, making the active state feel distinct without being loud. **Year Dividers** — A flex rule-label-rule with the year in `UnifrakturMaguntia` at `1.3rem` in `--red-dim`. Intentionally more muted than the `section-label` (which uses `--teal-lt` at `2rem`) so they read as internal list markers rather than page-level section headers. **Featured Entries** — Elevated with a left `2px solid var(--red-dim)` accent bar, a subtle warm wash background, `Cinzel Decorative` title, an excerpt in IM Fell italic, and the same `btn-read` style button from the index page. A `Spotlight` badge appears for editorially promoted entries. **Compact Entries** — A four-column flex row: Roman numeral · category col · title col · meta col. The columns are separated by very low-opacity `rgba` borders so the grid reads without being heavy. `Crimson Pro` for the title keeps density readable. On mobile the row wraps into two lines. **Pagination** — Uses the same cell-and-divider logic as the sub-nav and meta bar. A `page-spacer` flex element pushes the summary text and Next button to opposite ends, and the summary reads `Page 1 of 8 · 32 Chronicles` in italic IM Fell so it feels like a colophon. **Sidebar** — Browse by Discipline and Browse by Year both use lightweight `rgba` row dividers rather than full `--red-dim` borders, which keeps the sidebar feeling calmer than the dense listing column. The year bars are decorative width indicators. The WP sigil at the bottom uses the same inset-border panel treatment as the index page sidebar sigil.
